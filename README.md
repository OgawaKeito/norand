# norand（ロラネイター）

あなたはROLAND？ Yes / No で判定するインタラクティブWebアプリです。

## 概要

「norand」は、ROLANDという人物の基準に照らしてYes / Noで答えていく診断ツールです。
全画面表示のシンプルなUIで、直感的に操作できます。

## 技術スタック

- HTML5
- CSS3
- JavaScript（Vanilla JS）
- jQuery

## 特徴

- 全画面表示のシンプルなUI
- jQueryによるレスポンシブ対応
- シンプルなYes / Noの二択

## ファイル構成

```
norand/
├── index.html
├── main.js
├── style.css
├── README.md
└── img/
    └── ROLAND.png
```

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要です。
